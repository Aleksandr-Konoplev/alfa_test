### Эндпоинты:
Эндпоинты приведены к одной стилистике (`/api/v1/<model>/<action>/`) через переопределение 
в urlpatterns (products.api.v1.urls.py).  
Подробнее:  
`http://127.0.0.1:8000/redoc/`  
`http://127.0.0.1:8000/swagger/`
